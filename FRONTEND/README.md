# TikTok-UL (Avance 01) - Frontend
Integrantes: Mauricio Linares, Leonardo Sotelo, Diego Salazar, Alonso Murillo, Carlos Robles.

## Ejecutar local
1. npm install
2. npm run dev
3. Abrir http://localhost:5173

## Qué incluye este avance
- Login / Register (simulado, localStorage)
- Feed con likes persistentes
- Perfil y navegación básica
- Registro con verificación simulada (código)

## Qué falta (backend)
- Persistencia segura de contraseñas
- Upload de videos y almacenamiento
- APIs para likes/comments/streams
